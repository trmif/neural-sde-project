# Neural Stochastic Differential Equations

Курсовой проект по статье:

> Tzen, B., & Raginsky, M. (2019). **Neural Stochastic Differential Equations: Deep Latent Gaussian Models in the Diffusion Limit.** [arXiv:1905.09883](https://arxiv.org/abs/1905.09883)

## Содержание

- **КТ1** — описание проекта: постановка задачи, ключевые идеи статьи, цели
- **КТ2** — изучение подходов и работа с данными: обзор литературы, эксперименты, математика
